# Jeny

Jeny aims at being a simple yet powerful code generation and scaffolding
system.

## Command line

Jeny comes with a command line

```sh
jeny --help
```

### Generate a project from a scaffold

```sh
jeny -d name:hello g path/to/scaffold path/to/target
```
